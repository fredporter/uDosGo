# Demo media (optional)

Use this folder for **screenshots or short video frames** called out in [DEMO.md](../DEMO.md) (screenshot / video checklist).

- Filenames: descriptive, e.g. `thinui-feed.png`, `wp-restricted-logged-in.png`.
- **Do not commit large binaries** unless you explicitly want them in git; add patterns to `.gitignore` if you keep captures here locally.

Capturing is **optional** for repo health; the demo runs without files in this directory.
