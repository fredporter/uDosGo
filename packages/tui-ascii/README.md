# @udos/tui-ascii

**uDos TUI — ASCII animation toolkit (v1.0.0)** — spinners, loaders, splash templates, formula-driven line patterns, and Shell palette ids (`ascii:*`).

- **Spec:** [uDosDev `TUI_ASCII_ANIMATION_TOOLKIT_v1.0.0.md`](../../../uDosConnect/uDosDev/docs/specs/TUI_ASCII_ANIMATION_TOOLKIT_v1.0.0.md) (sibling checkout).
- **CLI:** `npm run build -w @udos/tui-ascii && node packages/tui-ascii/dist/cli/ascii-tool.js help`
- **Host API:** `GET /api/v1/meta/shell-palette` (uDosGo `@udos/host`) returns JSON including these commands.
